# 顧客カルテ管理 WebApp

美容室・サロン向けの顧客カルテ管理Webアプリです。  
Google Apps Script × Google スプレッドシートで動作し、月額費用ゼロで運用できます。
Built with assistance from Claude (Anthropic)
---

## 特徴

- **スマホ対応** — iPhone / Android のブラウザで動作。ホーム画面に追加してアプリ感覚で使えます
- **月額費用ゼロ** — データはご自身の Google スプレッドシートに保存されます
- **写真管理** — カメラ撮影またはライブラリから選択。自動で長辺800px・JPEG 70%に圧縮
- **絞り込み検索** — 名前（カタカナ）・施術メニュー・髪質でフィルタリング
- **来店回数管理** — あいうえお順 / 来店日順の切り替え表示
- **分析機能** — ロイヤルカスタマーランキング・平均単価・来店周期を自動集計
- **編集・削除** — 保存済みの施術記録をあとから修正・削除可能
- **設定カスタマイズ** — サロン名・メニュー項目を自由に変更

## セットアップ（所要時間：約15〜20分）

### 必要なもの

- Google アカウント（Gmail）
- パソコン（セットアップ時のみ。以降はスマホで運用可）

### 手順

**1. Google スプレッドシートを作成する**
1. [sheets.google.com](https://sheets.google.com) を開く
2. 新しいスプレッドシートを作成し、タイトルを「顧客カルテ」に変更する

**2. Apps Script を開く**
1. メニューの「拡張機能」→「Apps Script」をクリック
2. プロジェクト名を「顧客カルテ」に変更する

**3. Code.gs を設定する**
1. 左サイドバーの `Code.gs` を開く
2. 全選択して削除し、このリポジトリの `Code.gs` の内容を貼り付ける
3. Ctrl+S（Mac: Cmd+S）で保存

**4. index.html を追加する**
1. 左サイドバーの「ファイル」横の「＋」→「HTML」を選択
2. ファイル名を `index` にする（`.html` は自動付与）
3. 全選択して削除し、このリポジトリの `index.html` の内容を貼り付ける
4. Ctrl+S で保存

**5. 権限を付与する**
1. ドロップダウンで `initSheet` を選び、▶ 実行をクリック
2. 「承認が必要です」→「権限を確認」→ Google アカウントを選択
3. 「詳細」→「（プロジェクト名）に移動」→「許可」をクリック
4. 実行ログに「実行完了」と表示されれば OK（スプレッドシートに「カルテデータ」シートが作成されます）

**6. デプロイ（公開）する**
1. 右上の「デプロイ」→「新しいデプロイ」をクリック
2. 種類の歯車アイコン →「ウェブアプリ」を選択
3. 「次のユーザーとして実行」→「自分」、「アクセスできるユーザー」→「全員」に設定
4. 「デプロイ」をクリック
5. 表示された **ウェブアプリの URL** をコピーして保存

> ⚠️ コードを変更した際は必ず「新しいデプロイ」を作成し、新しい URL を使用してください。

**7. スマホのホーム画面に追加する**

- **iPhone（Safari）**：共有ボタン（□↑）→「ホーム画面に追加」
- **Android（Chrome）**：メニュー（︙）→「ホーム画面に追加」

## 機能詳細

### 検索
- 名前（カタカナ）の部分一致検索
- 施術メニュー・髪質ボタンで絞り込み（OR検索）
- 何も選ばず検索すると全件表示

### 顧客一覧
- あいうえお順 / 来店日順の切り替え
- 顧客名をタップすると全施術履歴を確認

### 施術記録追加
- メニュー・髪質をチップボタンで選択
- 金額はテンキーで入力
- 写真はカメラ撮影またはライブラリから選択（自動圧縮）
- 指名の有無、メモ・次回提案も記録可能

### 分析
- ロイヤルカスタマーランキング（来店回数 × 平均単価）
- お客様毎の平均単価ランキング
- お客様毎の来店周期（2回以上来店の顧客のみ）

### 設定
- サロン名の変更
- メニュー項目の追加・削除

## 注意事項

- データはご自身の Google アカウント内に保存されます。外部サーバーへの送信はありません
- 写真データはスプレッドシートのセルに直接保存されるため、枚数が増えると容量を消費します（Google Drive の空き容量をご確認ください）
- Google のサービス利用状況によっては、Google 側の料金が発生する場合があります（Drive 容量超過など）
- 第三者への再配布・販売は禁止しています
